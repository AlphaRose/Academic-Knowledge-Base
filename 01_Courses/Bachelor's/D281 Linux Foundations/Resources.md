# GNU & Linux Documentation
All of these are **extremely** wordy and are best for reference if you run across something you do not understand. 
- [GNU Core Utilities Documentation](https://www.gnu.org/software/coreutils/manual/coreutils.html)
- [Linux Filesystem Hierarchy Standard](https://refspecs.linuxfoundation.org/FHS_3.0/fhs/index.html)
- [Bash Reference Manual](https://www.gnu.org/software/bash/manual/bash.html)
- [Systemd Documentation](https://systemd.io/)
- [Man Pages](https://linux.die.net/man/)
